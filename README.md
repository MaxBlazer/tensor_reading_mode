# Режим чтения для интернет-статей 

Создаёт текстовый файл, содержащий текст статьи с веб-ресурса без лишнего "мусора".

## Первый запуск

1. Клонировать репозиторий
2. Установить зависимости:
```shell
pip install -r requirements.txt
```

## Использование

```shell
python3 main.py URL
```

Вместо URL подставить нужную ссылку.  
Результат сохраняется в `tensor_reading_mode/result.txt`, рядом со скриптом.

## Принцип работы

Из веб-страницы предварительно исключаются неинформативные теги (заданы имена этих тегов и шаблоны для поиска их по css-классам). 
Затем происходит поиск тела статьи по критерию максимального количества прямых потомков p-тегов.
Найденное тело ещё раз очищается (остаются лишь p и h теги), оборачивается в заданную ширину (80 символов) и сохраняется в файл.
